# To-Do List API 📝

A simple RESTful API built with **Node.js**, **Express**, and **MongoDB** to manage tasks.  
This project was created for learning purposes to practice backend development and CRUD operations.

---

## ⚙️ Features
- ➕ Add a new task  
- 📋 Get all tasks  
- 🔍 Get a task by ID  
- ✏️ Update an existing task  
- ❌ Delete a task  

---

## 🛠️ Technologies Used
- Node.js  
- Express.js  
- MongoDB + Mongoose
